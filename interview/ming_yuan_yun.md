#### 写出 HTTP 协议中你熟悉的几个状态码及其含义，499 是什么状态吗？

HTTP 响应状态码 指示特定 HTTP 请求是否已成功完成。

响应分为 5 类：
* 信息响应（100-199）
* 成功响应（200-299）
* 重定向（300-399）
* 客户端错误（400-499）
* 服务端错误（500-599）
