### golang 计算 【四舍五入】

go官方的math 包中提供了取整的方法，向上取整`math.Ceil()`，向下取整`math.Floor()`

golang没有类似python的`round()`函数，搜了一下很多都非常复杂，最后看到了一个清新脱俗的：先`+0.5`，然后向下取整！

```
package main
 
import (
    "fmt"
    "math"
)
 
func main() {
    var s float64
    for {
        _, err := fmt.Scanln(&s)
        if err != nil {
            break
        }
        fmt.Println(int(math.Floor(s+0.5)))
    }
}
```

### 计算一个整数在二进制形式下包含多少 1

```
package main
 
import "fmt"
 
func main() {
    var n,num int
    for {
        _, err := fmt.Scanln(&n)
        if err != nil {
            break
        }
         
        //n&(n-1) 可以消除n的最后一个1
        for n != 0 {
            n = n&(n-1)
            num++
        }
        fmt.Println(num)
    }
}
```
