## 面向对象

#### 什么是面向对象编程和面向对象编程语言？

- 面向对象编程, OOP , Object Oriented Programming
- 面向对象编程语言, OOPL, Object Oriented Programming Language
- 面向对象分析, OOA, Object Oriented Analysis
- 面向对象设计, OOD, Object Oriented Design


#### 面向对象编程两个非常基础的概念

- 类, class
- 对象, object


#### 面向对象编程的四大特性

- 封装
- 抽象
- 继承
- 多态
