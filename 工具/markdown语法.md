## 标题
在想要设置为标题的文字前面加 `#` 即可，
标准语法需要在 `#` 后面加一个空格
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
```

## 字体
要加粗的文字左右分别用两个 `*` 号括起来-(**加粗**)

要倾斜的文字左右分别用一个 `*` 号括起来-(*倾斜*)

要倾斜加粗的文字左右两别分别用三个 `*` 号括起来-(***加粗倾斜***)

要加删除线的文字左右两别用两个 `~` 号括起来-(~~删除线~~)
```
**加粗文字**
*倾斜文字*
***倾斜加粗文字***
~~有删除线文字~~
```

## 引用
在引用的内容前加 `>`

> 引用内容
>> 二次引用

```
> 引用内容
>> 引用内容 (引用嵌套)
```

## 分割线
三个或者三个以上的 `*` 

```
***
*****
```
***

## 图片
